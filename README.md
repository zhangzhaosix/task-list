# 任务清单 - 每日任务管理工具

一个简洁优雅的每日任务清单网页应用，帮助您高效管理日常任务。

## 功能特点

- ✅ 添加每日任务
- ✅ 标记任务完成状态
- ✅ 任务分类管理
- ✅ 本地数据持久化
- ✅ 简洁美观的界面设计
- ✅ 响应式布局，支持移动端

## 使用方法

直接打开 `index.html` 即可使用，无需安装任何依赖。

## 前端审查链路

项目内置 `.agents/skills/impeccable/`，用于前端审查与页面质量检查。

根目录下的 `PRODUCT.md` 为该链路的最小产品上下文文件，执行相关审查脚本前需要保留。
